App Name: MyFriends

User:
People who want to organize contacts in a more flexible way than a normal phonebook

Problem:
Default phonebooks only sort contacts alphabetically and don’t support custom grouping well

Core Idea:
Users can create folders (groups) and organize contacts inside those folders

MVP Features:
- Create folder (e.g. "School", "Gym", "Work")
- Add contact inside a folder (name, phone number)
- View contacts inside each folder
- Create nested folders (folder inside folder)
